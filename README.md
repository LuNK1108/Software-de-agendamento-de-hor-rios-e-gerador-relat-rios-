# Sistema de Agendamento de Horários e Gerador de Relatórios

O sistema **Lotus** é uma aplicação desenvolvida para auxiliar pacientes e profissionais da saúde no agendamento de consultas.

A plataforma permite que pacientes e profissionais possam:
- Agendar consultas
- Visualizar horários disponíveis
- Gerar e consultar relatórios
- Acessar prontuários
- Gerenciar receitas médicas

## Tecnologias utilizadas

- JavaScript
- PHP
- MySQL

## Contexto

Projeto desenvolvido como trabalho final da disciplina de **Experiência Criativa – 3º período**.
